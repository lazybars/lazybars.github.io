Lil Site

Goals

  • Create a hub for one's entire online presence
  • Make the sign up and customization process less than 2 minutes
  • Implement a CSS tiling system that can painlessly adapt to different media and device sizes
  • Make it epic - Check it out

Process

  • My research found that most link-in-profile tools like LinkTree and ContactInBio lack visuals and have 98% mobile users
  • I discovered it would be hard to retrieve thumbnail images from social media platforms because they frequently update their API
  • I switched profile creation method to form inputs resulting in a more manual approach that would not scale well and decided to put the project on hold

Tools

  • I designed the logo using Illustrator
  • I developed the website using HTML/CSS and some custom JS
  • I used Font Awesome for the web icons
  • I used Giphy for the gifs
  • I debugged with Chrome and Safari on desktop and mobile
