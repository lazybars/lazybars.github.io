The Challenge

Goals

  • Create a website for Dollarama to help boost sales from younger consumers
  • Build more of an online presence for Dollarama
  • Quickly funnel the consumers to their favourite products
  • Make it epic - Check it out

Process

  • My research found that incentivised challenges have a profound effect on young consumers
  • I began designing a website flow for the best mobile user experience
  • I created some flashy CSS/JS effects to excite and captivate users

Tools

  • I manipulated the images using Photoshop and created the logo using Illustrator
  • I developed the website using HTML/CSS, isotope.js, parrallax.js, tilt.js and custom JS
  • I used Font Awesome for the web icons
  • I debugged with Chrome and Safari on desktop and mobile
